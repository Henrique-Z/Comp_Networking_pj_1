|          Source Port           |        Destination Port        |    ↑
+--------------------------------+--------------------------------+    |

+-----------------------------------------------------------------+    | 20
|                      Acknowledgment Number                      |    | bits
+--------+-----------+-----------+--------------------------------+    |
|  Data  |           |U|A|P|R|S|F|                                |    |    ↑

|<4 bits>|<-6 bits-->|G|K|H|T|N|N|                                |    |    ↓

|            Checksum            |         Urgent Pointer***      |    ↓
+--------------------------------+----------------+---------------+
|                                                 |               |
≈                    Options**                    |    Padding    ≈
| <----------------- 0-40 bits -----------------> |               |
+-------------------------------------------------+---------------+
|                              data                               |

  * 4 bits to display the header lenght
 ** if data offset > 5: Padded at the end with "0" bytes if necessary
    variable - 8 bit interval
*** if URG flag bit is set to 1

ACK: Acknowledgment
